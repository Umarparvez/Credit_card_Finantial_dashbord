# 📊 Credit Card Financial Dashboard

An interactive Power BI dashboard analyzing credit card transactions, customer behavior, and financial performance.

---

##  Project Overview
This project focuses on analyzing credit card data to identify spending patterns, customer segments, and transaction trends. The dashboard helps in making data-driven business decisions.

---

##  Tools & Technologies
- Power BI  
- DAX (Data Analysis Expressions)  
- Excel / CSV  

---

##  Dataset
- credit_card.csv – transaction data  
- customer.csv – customer details  
- cc_add.csv – additional credit data  
- cust_add.csv – additional customer data  

---

##  Key Insights
- Identified high-value customers  
- Analyzed spending patterns across categories  
- Compared transaction volume vs revenue  
- Detected trends over time
